## 🎯 Sample Workflow

### Complete Patient Journey
```
1. Register Patient
   └─> Patient gets ID: P00001

2. Book Appointment
   └─> Select doctor, date, time
   └─> Appointment ID: A00001

3. Patient Arrives
   └─> Mark appointment as "In Progress"

4. Doctor Consultation
   └─> Create prescription
   └─> Record diagnosis, medicines
   └─> Generate PDF: RX00001
   └─> Appointment auto-marked "Completed"

5. Billing
   └─> Generate bill
   └─> Add consultation + lab + medicine charges
   └─> Apply discount if any
   └─> Print receipt: BILL00001

6. Follow-up
   └─> Schedule next appointment
   └─> Patient leaves with prescription and receipt
```

## 🆘 Need Help?

### Common Issues

**"Cannot connect to database"**
- Check SQL Server is running
- Verify credentials in `.env`
- Ensure database exists

**"Port 3000 already in use"**
- Change PORT in `.env` to 3001
- Or stop the other application

**"Module not found"**
- Run `npm install` again
- Check Node.js version (v16+)
